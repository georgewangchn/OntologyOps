# P6 · 多范式分层仲裁推理系统

> **多范式推理实战营 — 项目 6/7**
> 推理范式：**分层仲裁（P1-P5 融合）**
> P1-P3 确定性推理先行 → P4 模糊量化 → P5 概率校准 → 仲裁器裁决

## 分层架构

```
┌─────────────────────────────────────────┐
│         第一层：确定性推理                │
│  P1(OWL) + P2(Prolog) + P3(SPARQL)      │
│  一致确诊 → 直接输出                      │
│  一致排除 → 直接排除                      │
│  冲突 → 标记，进入第二层                   │
├─────────────────────────────────────────┤
│         第二层：模糊量化                  │
│  P4(Mamdani)                             │
│  置信度 > 0.7 → 升级高度疑似              │
│  置信度 < 0.3 → 降级低置信度              │
├─────────────────────────────────────────┤
│         第三层：概率校准                  │
│  P5(贝叶斯)                              │
│  后验概率作为最终排序依据                  │
├─────────────────────────────────────────┤
│         仲裁器                           │
│  确定性一致 → 采纳                        │
│  确定性冲突 → 概率优先                    │
│  模糊与概率一致 → 加权融合(0.4+0.6)       │
│  模糊与概率冲突 → 概率优先，标注冲突       │
└─────────────────────────────────────────┘
```

## 仲裁规则

| 确定性层 | 模糊层 | 概率层 | 仲裁结果 |
|---------|--------|--------|---------|
| 全部确诊 | - | - | 确诊（多引擎一致），置信度=1.0 |
| 全部排除 | - | - | 排除（多引擎一致），置信度=0.0 |
| 冲突 | - | 高概率 | 概率优先，标注冲突 |
| 部分一致 | 高 | 高 | 加权融合：概率×0.6 + 模糊×0.4 |
| 部分一致 | 高 | 低 | 冲突标注，概率优先 |
| 无结果 | 高 | 高 | 加权融合 |
| 无结果 | - | 高概率 | 仅概率推理 |
